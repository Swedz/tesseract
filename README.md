# Tesseract

Tesseract is a library that provides a smorgasbord of features to simplify the development process.

## Other Modules

Tesseract has other modules that can be used in specific contexts. This module is the "core" module that is included in
all others, and may be used in any project setting.

- [tesseract-neoforge](https://github.com/Swedz/tesseract-neoforge) (for the Minecraft mod loader [NeoForge](https://neoforged.net/))
